# 🏋️ Click FIT

> Less Input, More Motion.

A minimalist workout tracker designed for one-tap fitness logging.


⭐ Star this project if you like the idea.

## Why Click FIT?

I realized something strange.

The hardest part of working out wasn't the workout itself.

It was recording it.

Most fitness apps ask you to:

- Create workout plans
- Fill in forms
- Enter weights and reps
- Navigate through multiple screens

But when you're in the middle of your 5th set of deadlifts,
you don't want to type.

You just want to tap once and continue.

So I built Click FIT.
